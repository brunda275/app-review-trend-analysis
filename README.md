# App Review Trend Analysis AI Agent

## Overview
This project analyzes daily app reviews and identifies recurring issues,
requests, and feedback using a semantic embedding based agentic AI approach.

## Approach
- Reviews are processed as daily batches
- Semantic embeddings are used instead of traditional topic modeling
- Similar feedback is deduplicated into canonical topics
- A trend table is generated to track topic frequency over time
